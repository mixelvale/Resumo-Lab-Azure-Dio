# Resumo-Lab-Azure-Dio
Resumo dos primeiros passos e recursos dentro do MS Azure 
📍 O curso se refere a certificação AZ-900 do MS Azure, foram abordados conceitos iniciais dentro da plataforma, como criar sua conta gratuita, navegar pelo menus e os serviços disponibilizados na plataforma. 
📍Foram abordados temas relacionados a varias tecnologias que envolvem computação em núvem, ferramentas como Git, GitHub, versionamento de código e suas principais aplicações. 
📍Conceitos de nuvem pública e privada, preços e tipos de serviços. Comandos mais usados no Git, como criar repositório no GitHub e como trabalhar com repositórios locais e remotos. 
📍Configurar repositórios via Git, verififcar os hotóricos de alterações e remover alterações.
